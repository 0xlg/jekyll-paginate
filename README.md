# Jekyll::Generators::Pagination

Default pagination generator for Jekyll.

## Installation

Add this line to your application's Gemfile:

    gem 'jekyll-paginate'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jekyll-paginate
